#### 算法：
##### 问题转化：
```
对立：最小删除 --> 最多添加选择
差分：[a，b] --> <= b - <= a
降维：矩阵 -> 数组
```
##### 枚举、搜索、dp
```
**搜索**尽可能朝目标靠近，减小不必要的状态搜索，bidirectional bfs; 二分搜索; 将n个数分为k份，让每份的和的最大值最小，搜索时优先使k份平均
```
##### 出发点：
```
最小最大值
连续子数组：终点，差分枚举所有子数组
背包：第i个数选择与否
非连续子数组：第i个数作为终点
子区间: [0, i]
```
